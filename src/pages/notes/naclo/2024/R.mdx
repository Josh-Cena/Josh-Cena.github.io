In R1:

- (a) has all double vowels like "really", "soaked", "oasis" replaced with the first vowel; this is rule 4
- (b) has all "e" like "eggplant", "fuel", "efficiency" replaced with "a"; this is rule 5 (ignoring "e" before another vowel because "Earthling" remains unchanged)
- (c) has all double consonants like "carrot", "trillion", "dollar" replaced with single consonant; this is rule 2
- (d) has all final voiced consonants like "bad", "and", "lab" replaced with their voiceless counterparts; this is rule 1
- (e) is therefore rule 3: initial "s" replaced with "z".

All this is just to get familiar with the rule notation and mechanics. Now onto real business.

For R2, we just look at the last rule and see what language contains the feature that was generated by that rule. Language A's last change is (17) → ei, so it should contain some "ei" bigrams. Only Portuguese has them. Language C's last change is (23) → dd, so it should contain some "dd" bigrams. Only Sardinian has them. That leaves B as Italian.

- The first rule, (1)→∅ / \_#, deletes something at the end. The only thing that got deleted is "s" as in "nivis" and "cibus". Therefore (1) = s.
- The second rule, (2)→(3) / V_V, changes something between vowels—likely a consonant. This is b→u, as in cibu→civu and caballu→cavallu. It doesn't change the "b" in "lumbu" or "bonu" because they are not intervocalic.
- The third rule, (4)→(5) / \_t, changes something before "t". The only remaining change we haven't accounted for is ruptu → ruttu, so (4) = p, (5) = t.

The first rule that says change something into "e" should probably do that job. Indeed, all other "i" changed into "e" as well, such as siccu→secco, not just word initial. So (6) = i.

Other than long vowel to short vowel, there's another universal sound change: all "u" become "o", _except_:

- focu → fuoco (not foco)
- bonu → buono (not bono)
- sonāre → suonare (not sonare)
- lūna → luna (not lona)

Because we have "u" sounds that survive (especially "luna"), the rule u→o should come very early, so that later "u" can be reinserted by other rules. Therefore, (7) = u, (8) = o. Once we have this, we can apply the V[+L]→V[-L] rule to get the before/after of the (18)→(19) / (20) change.

Now we know the three rewrite rules that lead to Western Romance, we can use that as the starting point for deducing Portuguese. The e→∅ / (16) rule deletes some "e"s; the only "e"s that are deleted are the word-final ones such as "frecare" and "adottare". However, the "e" in "verde" or "neve" are not deleted; only those after "r" are. So Therefore (16) = r\_ (we don't have cases where "re" is not at the end, so I don't know if we also need # at the end).

Skimming the data, there are three changes:

- Voiceless stops become voiced stops, such as foco→fogo
- Vowels followed by "n" become nasal vowels, such as bono→bõo
- Double consonants become single consonants, such as cavallo→cavalo

We just need to devise the environment for each change.

- The stop voicing change doesn't change "frecare" into "vrecare", or "espata" into "esbata", so it only applies intervocalicly. It also doesn't make "rotto" become "rodo", so this change has to happen before the consonant deduplication. Therefore, (9) = [+Stop -Voiced], (10) = [+Stop +Voiced], (11) = V_V (since the \_V environment is not sufficient for this change)
- The Vn→Ṽ rule doesn't change "penna" into "pẽna" or "pẽa", which means that it has to come before consonant deduplication too, _and_ it should be restricted to the environment of pre-vowel, which does match the \_V condition given in the problem. Therefore, (12) = Vn, (13) = Ṽ, (14) = C\{2}, (15) = C\{1}.

Finally, we still need to figure out what (17) is in (17)→ei. Applying all previous rules, we have seno→sẽo, arena→arẽa. Therefore (17) = ẽ. Indeed, we have no instance of ẽ in the final data.
