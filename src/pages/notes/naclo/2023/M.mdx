To see the common parts, you have to squint a bit. For example, one might say that the shared part for "see" is "- Ēu-", but if you look closer, the preceding segment is either "l" or "l…ô". Schwas are often irrelevant to the meaning of the word and are inserted for phonotactics. (I've seen very few languages where schwas are differentiating.) So for now, I'll say that "see" is "l Ēu"/"l…ô Ēu". Similarly in "tie", all but one verb contain "k…ôlw…ô", while that one contains "kl…ôw", so this is the verb. In "sniff", the shared part is "jŇčo".

First look at the prefixes.

- "t…ô-" appears iff the subject is "I".
- "m…ôt…ô-" appears iff the subject is "we".
- "ine-" appears iff the object is "me".
- "ne-" appears iff the subject is "he", "they", or object is "us". This seems a bit awkward and messes up suffix analysis, so I'll talk about it later.

Next the suffixes:

- "-…£ Ēen" appears iff the object is "him". There are two cases where the "-…£ Ēen" is missing when the object is "him", but in both cases we have a "-rk…ôn" suffix, so "-…£ Ēen" has perhaps been absorbed into "-rk…ôn".
- "-t…ôk" appears iff the subject _or_ object is "you (pl.)"; interesting. The extra "-i" in "-it…ôk" appears iff we have "-rk…ôn" and this is also the case for "-im…ôk" and "-i…£…ôt", so we conclude that this "i" belongs to the progressive marker, which has the variant spelling "-rk…ôni".
- "-net" appears iff the object is "them" (the "n" disappears when preceded by "rk…ôn"), except in "you (pl.) + them" the "-net" is completely missing, so it may have been absorbed into "-t…ôk".
- "-…£…ôt" appears iff the object is "you (sg.)" (only one anyway).
- "-…£ Ēi" appears only when the subject is "you (sg.)", but in two other phrases with subject "you (sg.)", the suffix is something else. So it seems that "-…£ Ēi" means "you (sg.)" but can be replaced by "him" and "them" suffixes.

So this suggests that we have multiple morphemes that compete for the suffix position. In this data, we only know that you (pl.) > them/him > you (sg. subj.).

We have two morphemes left: ne- and -m…ôk. Note that "he + you (pl.)" and "you (pl.) + us" are both "ne-t…ôk", so there's a strong tendency to claim that "ne-" simultaneously stands for "he" and "us". Similarly, it could also stand for "they". However, in "he + us" = "ne-m…ôk", we need something to stand for "-m…ôk"! My proposal is that "ne-" stands for "he/they/us" and "-m…ôk" stands for "us", and is only there when something else wants to go in the prefix.

With this, M1 is doable:

a. k…ôl…ôw-net = tie-them = You (sg.) tie them up. ("them" wins over "you (sg. subj.)")  
b. m…ôt…ô-kl…ôw-…£…ôt = we-tie-you(sg.obj.) = We tie you (sg.) up.  
c. ine-k…ôlw…ô-rk…ôni-t…ôk = me-tie-prog-you(pl.) = You (pl.) are tying me up.  
d. ne-l Ēu-rk…ôni-t…ôk = he/they/us-see-prog-you(pl.) = He is seeing you (pl.) / They are seeing you (pl.) / You (pl.) are seeing us.

To translate back, we need to figure out some alternations we haven't accounted for, in particular "rk…ôn" vs. "rk…ôni", and which verb root to use (with different schwas).

- Progressive marker:
  - rk…ôni if followed by "…£…ôt", "m…ôk", "t…ôk". "rk…ôn" otherwise.
- Verb root:
  - l Ēu if preceded by a vowel, l…ô Ēu otherwise.
  - k…ôlw…ô if followed by two consonants, kl…ôw otherwise.
  - We can surmise that jŇčo follows the same schwa epenthesis rules as l Ēu, since they have the same CCV structure.

I got two phrases different, but I don't see how they can be remedied unless you buy the official solution's analysis that "ne-" means inverse alignment wholesale. But the official solution says that alternative analyses are possible, quoting the one I presented above, so I think multiple solutions are also accepted.

a. I tie him up = I-tie-him = t…ô-k…ôlw…ô-…£ Ēen  
b. You (sg.) are seeing us = <span style={{ color: "var(--color-red)" }}>us-see-prog-you(sg.subj.) = ne-l Ēu-rk…ôni-…£ Ēi</span> (this is different from the official solution, which is ne-l Ēu-rk…ôni-m…ôk. I don't consider mine wrong though. For one thing, we have no evidence of which of -…£ Ēi or -m…ôk takes priority.)  
c. You (pl.) sniff him = sniff-you(pl.) = j…ôŇčo-t…ôk ("you (pl.)" wins over "him")  
d. He is tying them up = he-tie-prog-them = ne-k…ôlw…ô-rk…ôn-et  
e. He sniffs me = <span style={{ color: "var(--color-red)" }}>me-sniff (he) = ine-jŇčo</span> (this is again different from the official solution of ine-jŇčo-…£ Ēi; if "me" = ine- and "he" = ne-, then we have no evidence of which one overrides the other.)
